Core Java Learning Repository
📌 Overview

This repository contains my hands-on practice and structured learning of Core Java concepts. It includes implementations of Object-Oriented Programming (OOP), collections, arrays, and basic Java Swing applications.

The project is organized into packages based on concepts for better clarity and understanding.

🎯 Learning Goals
Strengthen Core Java fundamentals
Master Object-Oriented Programming concepts
Practice real coding examples
Prepare for technical exams (e.g., Nepal Bank IT Exam)
Build a strong programming foundation for development
📂 Project Structure
src/
│
├── abstraction/                → Abstract classes and methods
├── abstractionExercise/        → Practice problems on abstraction
├── Aggregation/               → HAS-A relationship examples
├── arrays/                    → Array programs and logic building
├── collections/               → Java Collections (List, Set, Map)
├── Inheritance/               → IS-A relationship examples
├── javaSwing/                 → GUI-based applications using Swing
├── org/                       → Base or structured package (main classes)
├── overriding/                → Method overriding examples
├── OverridingExercise/        → Practice problems on overriding
├── practice_3/                → General practice programs
